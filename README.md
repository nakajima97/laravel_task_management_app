# 概要

Laravel でタスク管理アプリを作成した

# 作成目的

-   laravel の基本機能を実践する
    -   データの CRUD
    -   Laravel Breeze を用いたログイン機能の実装
    -   単体テスト
    -   ブラウザテスト
-   ユーザ種別によってログイン後の画面を変更する方法の学習
-   Laravel の Linter・Formatter の使い方の学習

# 使用技術

-   Laravel 9
-   Laravel Sail
-   Laravel Breeze
